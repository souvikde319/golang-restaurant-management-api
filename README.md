# 📖 Overview

Restaurant Management API is a backend application developed using **Golang**, **Gin**, **GORM**, and **MySQL**.

It provides REST APIs for:

- 👤 User Management
- 🍔 Food Management
- 📋 Menu Management
- 🪑 Table Management
- 🛒 Order Management
- 📦 Order Item Management
- 🧾 Invoice Management
- 🔐 JWT Authentication
